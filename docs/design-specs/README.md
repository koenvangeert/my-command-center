# AI Command Center — Design Specifications

This directory contains detailed design prompts for the three main pages of the AI Command Center app. Each document describes every feature, UI element, and interaction on its respective page, intended as a comprehensive reference for generating UI designs.

## Pages

1. **[General Board](./general-board.md)** — The kanban-style task management board (default view)
2. **[Task View](./task-view.md)** — The detailed task view with agent terminal and task info panel
3. **[Self-Review](./self-review.md)** — The code review view with diff viewer, file tree, and comment sidebar
